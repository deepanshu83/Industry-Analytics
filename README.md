# 📊 Retail Sales Intelligence Platform

An end-to-end Data Analytics project built using Python, SQL, and Power BI to analyze retail sales data and generate actionable business insights.

---

## 🚀 Project Overview

This project demonstrates the complete Data Analytics workflow used in industry.

Workflow:

Raw Dataset
→ Data Cleaning
→ Exploratory Data Analysis (EDA)
→ SQL Analysis
→ Interactive Power BI Dashboard
→ Business Insights
